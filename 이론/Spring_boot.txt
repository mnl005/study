스프링부트는 단순히 실행되고, 프로덕션 제품 수준의 스프링 기반 어플리케이션을 쉽게 만들 수 있다
스프링 부트 어플리케이션에는 스프링 구성이 거의 필요치 않다
스프링 부트 java-jar로 실행하는 java 어플리케이션을 만들 수 있다



- 스프링 부트의 주요 목표 -

1. 스프링 개발에 대한 빠르고, 광범위하게 적용 가능한 환경
2. 기본값 설정이 있으나 수정가능
3. 대규모 프로젝트에 공통적 비 기능 제공(보안,모니터링)
4. XML 구성 요구사항이 전혀 없음(개꿀)


- 툴 -

빌드 : Maven, Gradle
Servlet Containers : Tomcat, Jetty, Unertow
